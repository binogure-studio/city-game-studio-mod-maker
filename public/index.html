City Game Studio - Mod Maker v1.0

Mod Maker helps you generate a zip file that contains all the required files to make a mod. Once ready, click on the Download button, and save the file on your hard drive. Then extract mod.zip into a new folder. Launch City Game Studio, create a new mod, and select the created folder. Click on Create and your mod is now ready to be used.
